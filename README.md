
                                                     🔍 IPTV Channel Checker (VLC Mode) v3.0

                                                     
أداة احترافية مبنية بلغة Python لفحص روابط قنوات IPTV (ملفات M3U) بدقة عالية. يتميز هذا الإصدار بمحاكاة كاملة لسلوك مشغل VLC لضمان تخطي الحمايات والحصول على نتائج فحص حقيقية.

🌟 المميزات الرئيسية
محاكاة VLC الكاملة: يستخدم User-Agent الخاص بـ VLC ويتعامل مع ترويسات الاتصال بنفس الطريقة.

دعم خيارات EXTVLCOPT: تحليل وتطبيق خيارات VLC المتقدمة الموجودة داخل ملفات M3U (مثل Referrer و User-Agent الخاص بكل قناة).

فحص ذكي للمحتوى: لا يعتمد فقط على استجابة السيرفر، بل يقوم بقراءة أول 2048 بايت للتحقق من بصمة البث (MPEG-TS أو M3U8).

سرعة فائقة: يعتمد على البرمجة غير المتزامنة (asyncio و aiohttp) لفحص مئات القنوات في ثوانٍ معدودة.

واجهة رسومية (GUI): واجهة مستخدم بديهية مبنية بـ Tkinter مع نظام تصفية وبحث متقدم.

تصدير النتائج: دعم تصدير القنوات الشغالة إلى ملفات M3U جاهزة للتشغيل أو ملفات JSON.
